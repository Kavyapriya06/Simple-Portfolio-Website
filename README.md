# Simple-portfolio-website
Simple portfolio website developed with HTML and basic CSS.

The project was created to strengthen my web development fundamentals and to provide a professional platform for presenting my qualifications and accomplishments.

1.Technologies Used
i.HTML5
ii.CSS3
iii.Google Fonts (Poppins)
iv.Font Awesome Icons

2.Project Structure
i.index.html – Main portfolio page
ii.contact.html – Contact page
iii.style.css – Styling for the website

3.Pages
i.Home Page: Contains my introduction, career objective, education details, technical skills, achievements, internship experience, projects, and contact information.
ii.Contact Page: Includes a contact form for visitors to send messages and provides links to my LinkedIn and GitHub profiles.

4.Features
i.Simple and responsive design
ii.Professional portfolio layout
iii.Easy navigation between pages
iv.Contact form
v.Social media integration
